IDZIAK SKP – paczka do GitHub Pages

Pliki gotowe do podmiany na GitHub.

Zawartość:
- index.html
- styles.css
- script.js
- polityka-prywatnosci.html
- 404.html
- assets/ (obrazy, logo, favicon)
- CNAME, robots.txt, sitemap.xml, llms.txt, manifest.webmanifest

Jak wgrać:
1. Rozpakuj ZIP.
2. Na GitHub wejdź do repozytorium strony.
3. Kliknij Add file → Upload files.
4. Wgraj wszystkie pliki z paczki i potwierdź podmianę starych plików.
5. Kliknij Commit changes.
6. Odśwież stronę po kilku minutach przez Ctrl+F5.
